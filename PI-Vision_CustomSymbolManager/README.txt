To generate a new Referential_PICustomSymbolFiles as input file :
- Use PIVision-Export-Symbols-Hash.ps1 script to generate the file.
- Edit it and add manually a new column to mention the symbol name for each files (duplicate the lines if a file is used for multiple symbols).

To run an analysis, use PICustomSymbolsManager.ps1 script.
If you want to modify the parameters, please look the Header of the script to get their description.
